1.7注释

1、Python注释

为什么要有注释： 注释可以起到一个备注的作用，这个方法函数，变量到底是干嘛用的，如果没有注释时间长了即使是自己可能都不知道这代码到底是干嘛用的。所以注释起到的作用就是方便自己查看写过的代码 ，别人来接手你的代码能看懂。

来看下有注释跟没注释的区别就明白为啥第一天就强调注释的重要性了。

3、单行注释：

python中单行注释用#号，#号右边的就是注释的内容，python解析器遇到#号就会当做注释，不会去解析#号后面的内容。

4、多行注释：

多行注释使用三个单引号'''内容''' 也可以使用三个双引号"""内容"""

'''
三对英文状态下的单引号，python多行注释
三对英文状态下的单引号，python多行注释
三对英文状态下的单引号，python多行注释
三对英文状态下的单引号，python多行注释
'''

"""
三对英文状态下的双引号，python多行注释
三对英文状态下的双引号，python多行注释
三对英文状态下的双引号，python多行注释
三对英文状态下的双引号，python多行注释
"""

5、特殊注释：

Python中有两句特殊的注释，在py文件首行 #!/usr/bin/env python3 ; # -- coding=utf-8 -- 这两句注释分别指的是指定python解析器的路径，指定编码格式，只能写在py文件最前面。
