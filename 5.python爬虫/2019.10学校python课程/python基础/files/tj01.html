Python基础

if-else语句

1、条件判断语句

Python条件语句是通过一条或多条语句的执行结果（True或者False）来决定执行的代码块。

2、if-else 语句

在生活中我们遇到问题也经常需要一个条件来判断是否去做一件事。

比如： 如果明天不下雨，我们就去爬山，否则打羽毛球。

伪代码实现：

if 明天不下雨：
    我们去爬山
else:
    我们去打羽毛球

3、猜数字游戏

"""
猜数字游戏
"""
# 接收玩家输入数字
num = input('你觉得现在小明想的数字是什么：')

# 定义一个变量，表示小明心中所想
num1 = 5 

if num == num1:  # 条件判断
    print('恭喜你猜对了，看来你们是好基友心有灵犀')   # 条件成立打印这句话
else:
    print('很遗憾，你居然不知道你基友心中所想')   #  条件不成立打印这句话
